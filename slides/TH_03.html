Thực hành: Tạo ứng dụng mobile ATM

  Tạo ứng dụng trên laptop

Command Prompt


1. Cài đặt Node.js và thư viện Ionic, Cordova
https://nodejs.org/
npm install -g ionic cordova

2. Cài đặt Sublime Text 3
https://www.sublimetext.com/3

3. Tạo dự án
ionic start ATM blank
cd ATM

4. Chạy ứng dụng
ionic serve

5. Đổi mobile display
Press Ctrl-Shift-I
  • Cài đặt Ionic
  • Tạo ATM
  • Chạy thử trên laptop

  Chạy ứng dụng trên thiết bị Android

Command Prompt


1. Ngừng ứng dụng 
Nhấn Ctrl-C

2. Cài hệ điều hành Android
ionic cordova platform add android

3. Kết nối thiết bị
  . nối dây vào cổng USB
  . chuyển thiết bị vào chế độ Debug
  Settings->Developer options->USB debugging

4. Chạy ATM trên thiết bị
ionic cordova run android

5. Tên trên thiết bị: MyApp
  • Cài Android
  • Chạy trên thiết bị

  MyApp trên thiết bị

  • Ionic Blank
